# Edge Functions

Supabase Edge Functions (Deno) for serverless tasks:

- `ocr-process/` — async OCR processing via Mistral API
- `bridge-webhook/` — incoming transactions from Bridge (PSD2)
- `nightly-recompute/` — full state recompute via pg_cron

See [Supabase Edge Functions docs](https://supabase.com/docs/guides/functions).
